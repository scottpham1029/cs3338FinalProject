# Mock Plugin Source Directory

This folder represents the planned source code location for the 
HDR Imagery Dynamic Tone Mapping OpenFX plugin.

In a full implementation, this directory would contain C++ 
source files, OpenFX SDK integration files, CMake build 
configuration, tone mapping algorithms, and DaVinci Resolve 
plugin packaging files.

For this course project, the folder is used to document the 
expected development structure rather than provide 
a working plugin implementation.
